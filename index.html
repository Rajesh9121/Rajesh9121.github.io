Intro

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

Get Started

Python is an interpreted programming language, this means that as a developer you write (.py) files

Syntex

Indentation refers to the spaces at the beginning of a code line.

Comments

Comments can be used to explain code.Comments can be used to make the code more readable.

Variables

Variables are containers for storing data values.

Data Types

Variables can store data of different types, and different types can do different things.

Numbers

There are three numeric types in : int, float, complex.

Casting

There may be times when you want to specify a type on to a variable. This can be done with casting.

Strings

Strings in are surrounded by either single quotation marks, or double quotation marks.

Booleans

Booleans represent one of two values: True or False.

Operators

Operators are used to perform operations on variables and values.

Lists

Lists are used to store multiple items in a single variable.

Tuples

Tuples are used to store multiple items in a single variable.

Sets

A set is a collection which is unordered, unchangeable*, and unindexed.

Dictionaries

A dictionary is a collection which is ordered*, changeable and do not allow duplicates.

If..Else

These conditions can be used in several ways, most commonly in "if statements" and loops.

While Loops

while loop we can execute a set of statements as long as a condition is true.

For Loops

A for loop is used for iterating over a sequence

Functions

A function is a block of code which only runs when it is called.

Lambda

A lambda function is a small anonymous function.

Arrays

Arrays are used to store multiple values in one single variable

Classes/Objects

Almost everything in Python is an object, with its properties and methods.

Inheritance

Inheritance allows us to define a class that inherits all the methods and properties from another class.

Iterators

An iterator is an object that contains a countable number of values.

Scope

A variable is only available from inside the region it is created. This is called scope.

Modules

A file containing a set of functions you want to include in your application.

Math

Python has a set of built-in math functions, including an extensive math module.

JSON

JSON is a syntax for storing and exchanging data.

RegEx

A RegEx, or Regular Expression, is a sequence of characters that forms a search pattern.

PIP

PIP is a package manager for Python packages, or modules if you like.

Try Except

The try block lets you test a block of code for errors. The except block lets you handle the error.

User Input

Python stops executing when it comes to the input() function, and continues when the user has given some input.

String Formating

To make sure a string will display as expected, we can format the result with the format() method.
